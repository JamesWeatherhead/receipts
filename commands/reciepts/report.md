# reciepts:report

Aggregate all verdicts into a final report.

## Your Task

1. **Read all verdict.md files** in `<path>/verdicts/`
2. **Categorize by status**: VALID, ADJUST, INVALID, UNCLEAR
3. **Generate RECIEPTS.md** in the paper folder

## Output Format

Write to `<path>/RECIEPTS.md`:

```markdown
# Citation Verification Report

**Document:** [filename]
**Verified:** [date]
**References checked:** [N]
**References excluded:** [Y]

## Results Summary

| Status | Count | Percentage |
|--------|-------|------------|
| VALID | X | X% |
| ADJUST | Y | Y% |
| INVALID | Z | Z% |
| UNCLEAR | W | W% |

---

## Invalid Citations

[For each INVALID verdict:]

### Reference [N]: [Short citation]

**Status:** INVALID

**Claim in manuscript:**
> [verbatim quote]

**What source actually states:**
> [verbatim quote]

**Problem:** [description]

**Required fix:** [specific correction]

---

## Citations Needing Adjustment

[For each ADJUST verdict:]

### Reference [N]: [Short citation]

**Status:** ADJUST

**Claim in manuscript:**
> [verbatim quote]

**What source actually states:**
> [verbatim quote]

**Discrepancy:** [specific difference]

**Required fix:** [specific correction]

---

## Valid Citations

[X] citations accurately represent their sources.

<details>
<summary>View details</summary>

| Ref | Citation | Instances |
|-----|----------|-----------|
| [1] | Author et al. (Year) | N |
| ... | ... | ... |

</details>

---

## Unclear Citations

[For each UNCLEAR verdict, explain why verification was not possible]

---

## Full Verdict Files

- [Reference 1](verdicts/ref_01/verdict.md)
- [Reference 2](verdicts/ref_02/verdict.md)
- ...

---

*Generated by reciepts*
```

## After Generating

Display summary to user:
- Total verified
- Invalid count (highlight if > 0)
- Adjust count
- Path to full report
